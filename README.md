# Simple Calculator

## How to run the app
1. **Clone the repository**: 

   `git clone https://github.com/Julianioorio/Calculator.git`
2. **Go to the project directory**:  

   `cd Calculator`
3. **Install dependencies**:  

   `npm install` or `yarn install` or `pnpm install`
4. **Launch a project**: 

   `npm run dev` 
5. **build**:

   `npm run build`

## 1. Root Folders and Files

1. **root/**  
   Compiled project build generated by Webpack. Do **not** edit manually.  

2. **git/**  
   Git service data.  

3. **git-hooks/pre-commit**  
   Hook executed before each commit (e.g., to run a linter).  

4. **node_modules/**  
   Installed npm dependencies. Automatically generated.  

5. **.gitignore**  
   Lists files and folders ignored by Git.  

6. **eslint.config.mjs**  
   ESLint configuration file.  

7. **package.json**  
   Contains project dependencies, library versions, npm scripts, and project metadata.  

8. **package-lock.json**  
   Locks exact versions of all installed dependencies.  

9. **webpack.config.js**  
   Webpack configuration: entry points, SCSS/JS processing, plugins, build output to `root/`.  

10. **README.md**  
    Project documentation.  


## 2. src Folder — Source Code

1. **Js Folder — JavaScript Logic**  
   - **calc.js** — Core calculation logic.  
   - **main.js** — File that imports and connects all JavaScript modules.  
   - **num.js** — Handles numeric input.  
   - **otherCalc.js** — Additional mathematical functions.  
   - **phantomNum.js** — Calculation history.  

   #### Theme Subfolder
   - **themechenge.js** — Theme switching (light/dark mode).  

2. **SCSS Folder — Styles**  
   - **_mixins.scss** — SCSS mixins.  
   - **main.scss** — File that imports all SCSS components.  
   - **normolize.scss** — CSS normalization.  

   #### Components Subfolder
   - **buttons.scss** — Button styles.  
   - **calculator.scss** — Calculator block styles.  
   - **display.scss** — Display/screen styles.
