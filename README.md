### Project Structure

---

## 1. Root Folders and Files

1. **root/**  
   Compiled project build generated by Webpack. Do **not** edit manually.  

2. **git/**  
   Git service data.  

3. **git-hooks/pre-commit**  
   Hook executed before each commit (e.g., to run a linter).  

4. **node_modules/**  
   Installed npm dependencies. Automatically generated.  

5. **.gitignore**  
   Lists files and folders ignored by Git.  

6. **eslint.config.mjs**  
   ESLint configuration file.  

7. **package.json**  
   Contains project dependencies, library versions, npm scripts, and project metadata.  

8. **package-lock.json**  
   Locks exact versions of all installed dependencies.  

9. **webpack.config.js**  
   Webpack configuration: entry points, SCSS/JS processing, plugins, build output to `root/`.  

10. **README.md**  
    Project documentation.
